# クイズアプリ

学校で学習したHTML、CSS、JavaScript、JSONを使用して作成した簡易的なWebアプリです。

## 内容

このアプリでは、以下の2種類のクイズを遊ぶことができます。

- 県庁所在地クイズ
- 世界の首都クイズ

トップページから遊びたいクイズを選択できます。

## 機能

- JSONファイルから問題データを読み込み
- 地方ごと、地域ごとに問題を選択
- 選択肢のランダム表示
- 問題のランダム出題
- 制限時間付きクイズ
- 選択した時点で正解・不正解を表示
- 結果画面で正解数と正答率を表示

## 使用技術

- HTML
- CSS
- JavaScript
- JSON
